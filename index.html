DBMS (Database Management System)

DBMS is a computerized record-keeping system. It is a software that defines, manipulates, and manages the database. It allows users to access files, update records, and retrieve data as requested.

Advantages of DBMS

  • Sharing Data
  • Reduce Data Redundancy
  • Data Backup and Recovery
  • Data Security
  • Multiple User Interface
  • Data Integrity

Disadvantages of DBMS

  • Expensive
  • Changing Technology
  • Backup is needed

Database Models

Hierarchical Database Model

This is one of the oldest database models, where data is represented in the form of records organized in a tree-like structure.

Network Database Model

The network model was designed to address the limitations of the hierarchical model by allowing many-to-many relationships.

Relational Database Model

In this model, data are organized into tables that contain rows and columns, also known as relations. It is the most common database model used today.

Structured Query Language (SQL)

SQL is a standard language for accessing and managing data in a database. It includes sub-languages such as DDL, DML, and DCL.

Normalization

Normalization is the process of breaking down complex relations into simpler relations to reduce redundancy and ensure data integrity.

First Normal Form (1NF)

1NF eliminates duplicate columns and ensures each table has a primary key.

Second Normal Form (2NF)

2NF removes partial dependencies by placing data that applies to multiple rows in separate tables.

Third Normal Form (3NF)

3NF removes columns that are not dependent on the primary key and ensures that data is fully normalized.
